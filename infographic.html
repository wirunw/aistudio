สร้าง Dashboard อย่างมืออาชีพด้วย AI

สรุปหลักการ, เครื่องมือ, และ Prompt จากคลาส Business Intelligence

หลักการ: ทำไมต้องใช้ AI ช่วยออกแบบ Dashboard?

ในยุค Data-Driven การมี Dashboard ที่ดีคือหัวใจสำคัญ แต่การออกแบบ Dashboard ที่ตอบโจทย์ผู้บริหารมักใช้เวลานานและสื่อสารไม่ตรงกัน การใช้ Generative AI (เช่น Gemini, ChatGPT) เข้ามาช่วยเปรียบเสมือนการมี "นักออกแบบต้นแบบ (Prototype Designer) ส่วนตัว"

  • ลดเวลา: สร้างภาพร่าง (Mockup) คุณภาพสูงได้ในไม่กี่วินาที
  • สื่อสารชัดเจน: แปลง "ความต้องการ" (เช่น KPIs, ตัวชี้วัด) ให้กลายเป็น "ภาพ" ที่ทุกคนเห็นตรงกัน
  • ทดลองได้ทันที: ช่วยให้เราสามารถปรับเปลี่ยน Layout หรือชนิดกราฟได้หลากหลาย จนกว่าจะเจอแบบที่เหมาะสมที่สุดก่อนลงมือสร้างจริง

(เช่น ผู้บริหาร, ทีมแพทย์)

3. เลือกข้อมูล (KPIs)

ต้องการดูตัวชี้วัดอะไรบ้าง
(เช่น อัตราครองเตียง, รายได้)

ระบุ Theme, Layout, และคำสั่ง
(เช่น "สร้าง...สไตล์ Modern")

1. ตารางสรุปเครื่องมือและ Prompt สำหรับสร้าง Dashboard

หัวข้อ (Topic) รายละเอียด / คำสั่ง (Description / Prompt) แหล่งที่มา (Source)
เครื่องมือสร้าง Prompt เว็บแอปฯ ช่วยร่าง Prompt เพื่อสั่ง AI สร้างภาพ Dashboard Design
https://short.bdi.or.th/genPrompt 		BDI
โครงสร้าง Prompt (Template) 
1. จุดประสงค์: (เช่น ติดตามยอดขาย)
2. ผู้ใช้งาน: (เช่น ผอ.โรงพยาบาล, CFO)
3. ข้อมูล: (KPIs, กราฟที่ต้องการ)
4. Layout: (เช่น 3x3 grid, หน้าเดียวจบ)
5. Theme: (เช่น สีฟ้า-ขาว, Professional)
6. คำสั่ง: "Create a professional executive dashboard design..."
 BDI
ตัวอย่าง Prompt (โรงพยาบาล) 
CRITICAL: Image aspect ratio must be 16:9.
Create a high-resolution, professional medical executive dashboard design.
The design must be clean, modern, and easy to read, suitable for hospital executives.
It should look like a real, functional application screenshot.
CRITICAL: Use ONLY English language for all text labels, titles, and data descriptions.
Dashboard Title: "Healthcare Executive Dashboard"
KPIs: Bed Occupancy Rate, Total Revenue, Patient Satisfaction
Charts: Show KPIs as cards, a line chart for revenue trend, and a table for AI alerts...
